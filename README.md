## Run Locally

Clone the project

```bash
  git clone https://github.com/mkoroleva5/online-store.git
```

Go to the project directory

```bash
  cd online-store
```

Install dependencies

```bash
  npm i
```

Start the server

```bash
  npm run dev
```

## Build

To build this project run

```bash
  npm run build
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
